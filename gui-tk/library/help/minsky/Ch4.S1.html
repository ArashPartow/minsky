4.1 Operations

add +

Add multiple numbers together. The input ports allow multiple wires, which are all summed. If an input port is unwired, it is equivalent to setting it to zero.

subtract --

Subtract two numbers. The input ports allow multiple wires, which are summed prior to the subtraction being carried out. If an input port is unwired, it is equivalent to setting it to zero. Note the small ‘+’ and ‘-’ signs on the input port indicating which terms are aded or subtracted from the result.

multiply ×\times

Multiple numbers with each other. The input ports allow multiple wires, which are all multiplied together. If an input port is unwired, it is equivalent to setting it to one.

divide ÷\div

Divide a number by another. The input ports allow multiple wires, which are multiplied together prior to the division being carried out. If an input port is unwired, it is equivalent to setting it to one. Note the small ‘×\times’ and ‘÷\div’ signs indicating which port refers to the numerator and which the denominator.

log

Take the logarithm of the xx input port, to base bb. The base bb needs to be specified — if the natural logarithm is desired (b=eb=e), use the ln operator instead.

pow xyx^{y}

Raise one number to the power of another. The ports are labelled xx and yy, referring the the formula xyx^{y}.

lt <<

Returns 0 or 1, depending on whether x<yx<y is true or false.

le ≤\leq

Returns 0 or 1, depending on whether x≤yx\leq y is true or false.

eq ==

Returns 0 or 1, depending on whether x=yx=y is true or false.

min

Returns the minimum of xx and yy.

max

Returns the maximum of xx and yy.

and ∧\wedge

Logical and of xx and yy, where x<0.5x<0.5 means false, and x>0.5x>0.5 means true. The output is 1 or 0, depending on the result being true or false respectively.

or ∨\vee

Logical or of xx and yy, where x<0.5x<0.5 means false, and x>0.5x>0.5 means true. The output is 1 or 0, depending on the result being true or false respectively.

not ¬\neg

The output is 1 or 0, depending on whether x<0.5x<0.5 is true or false respectively.

time

Returns the current value of system time.

copy

This just copies its input to its output, which is redundant on wiring diagrams, but is needed for internal purposes.

integrate

Creates an integration (or stock) variable. Editable attributes include the variable’s name and its initial value at t=0t=0.

data

A data interpolation widget. Currently, the data must be imported from a file containing two values on each line, eg:

0.1 0.3
0.5 0.7
0.9 1
If the input is less than the minimum key value (0.1 here), then the operation outputs the corresponding value (0.3). Similarly if the input is greater than the maximum (0.9), the corresponding value (1) is output. If it lies in between two keys (eg 0.2), the the output is linearly interpolated (0.4).

More formally, a data block is an empirical function, based on a table of pairs of values (xi,yi,i=1⁢…⁢n,xi+1>xix_{i},y_{i},i=1\ldots n,x_{i+1}>x_{i}) read in from a file. The function’s output is linearly interpolated from the data, ie:
