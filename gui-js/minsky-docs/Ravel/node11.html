Options

The options menu allows you to customise aspects of Ravel and Minsky. At the moment most options pertain to Minsky rather than Ravel, but this will change as Ravel increases in complexity. In this section of the manual we ignore options pertaining only to Minsky; these are covered in Introduction to Minsky

Preferences
  • Number of recent files to display -- this determines how many previously edited files are displayed on the Recent files menu.
  • Wrap long equations in LATEX export. If ticked, Ravel & Minsky will use the LATEX ``breqn'' package to produce nicer looking automatically line-wrapped formulae. Because not all LATEX implementations are guaranteed to support breqn, untick this option if you encounter problems.
  • select a font for variable names, parameter names, etc.
Background colour
-- select a colour for the wiring canvas. The default is white.
