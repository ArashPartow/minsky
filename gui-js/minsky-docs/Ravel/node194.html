File Menu Minsky commands

Library
Opens a repository of models for the Minsky simulation system.
Recording
Record the states of a model as it is being built for later replay. This is useful for demonstrating how to build a model, but bear in mind that recorded logs are not, in general, portable between versions of Minsky.
Replay recording
Replay a recording of model states.
