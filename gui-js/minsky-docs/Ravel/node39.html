Linking Ravels

Ravels which have the same dimensions can be linked to each other. For example, one Ravel may contain inflation data by country by month, while another contains unemployment data by country by month. If the two or more Ravels are linked, then selections applied to one Ravel are applied to the other, which makes it easy to compare these variables by country over time. For more details see Linking Ravels.
